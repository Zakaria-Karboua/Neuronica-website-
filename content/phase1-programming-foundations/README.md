# AI Engineering Curriculum — 2026 Edition

A complete, from-first-principles path from programming fundamentals to production AI systems: how AI works internally, how to build it from scratch, and how to ship it.

**Format:** every topic is a standalone Markdown lesson following the same 17-section template (Introduction → Theory → Math → Algorithm → Python → From Scratch → Library Version → Visuals → Examples → Industry Use Cases → Common Mistakes → Best Practices 2026 → Exercises → Mini Project → Interview Prep → Summary → References).

**Delivery model:** built phase by phase. You review each phase before the next is generated.
